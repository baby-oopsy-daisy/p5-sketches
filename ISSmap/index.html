International Space Station Live Path Trace

The sketch here traces the path taken by the ISS(International Space Station).

    The Sketch uses three API's

  1. MAPBOX : For loading the map
  2. OPEN NOTIFY : For the position data of ISS (in Longitude & Latitude)
  3. CORS-anywhere : To handle cross-origin requests

The location is updated at an interval of around 1 - 1.3 minutes.
